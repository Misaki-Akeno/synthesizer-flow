# Synthesizer Flow 模块文档

本文档列出了 Synthesizer Flow 平台中目前可用的所有音频合成与处理模块。

## 信号源 (Signal Sources)

### 1. 简单振荡器 (Simple Oscillator)
- **类型标识**: `simpleoscillator`
- **描述**: 生成基本波形的振荡器，提供频率和波形控制。它是最基础的声音生成单元。

### 2. 高级振荡器 (Advanced Oscillator)
- **类型标识**: `advancedoscillator`
- **描述**: 支持复音、包络控制（ADSR）和更多波形选项的高级振荡器。相比简单振荡器，它能创造出更丰富多变的声音。

### 3. 小号合成器 (Trumpet Synthesizer)
- **类型标识**: `trumpet`
- **描述**: 模拟小号音色的单音合成模块，带有滑音和表现力控制。专门用于生成类似铜管乐器的声音。

## 输入 (Inputs)

### 4. MIDI 输入器 (MIDI Input)
- **类型标识**: `midiinput`
- **描述**: 接收并处理外部 MIDI 控制器（如 MIDI 键盘）的输入信号。允许用户使用硬件设备控制合成器。

### 5. 键盘输入器 (Keyboard Input)
- **类型标识**: `keyboardinput`
- **描述**: 提供屏幕虚拟钢琴键盘，同时也支持使用计算机键盘进行演奏。方便没有外部设备的用户进行创作。

## 调制 (Modulation)

### 6. 低频振荡器 (LFO)
- **类型标识**: `lfo`
- **描述**: 产生低频调制信号（通常低于人耳听觉范围），用于调制其他模块的参数（如振荡器的频率或滤波器的截止频率），产生颤音、哇音等效果。

## 效果 (Effects)

### 7. 混响效果 (Reverb)
- **类型标识**: `reverb`
- **描述**: 添加空间混响效果到输入的音频信号。通过模拟声音在空间中的反射，使声音听起来更加饱满和有空间感。

## 输出 (Outputs)

### 8. 扬声器 (Speaker)
- **类型标识**: `speaker`
- **描述**: 音频输出模块，接收音频信号并输出到系统扬声器。这是音频流的终点。
